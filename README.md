🌊 ענקי המים (OFR) - האנציקלופדיה המודולרית

ברוכים הבאים לפרויקט ענקי המים (Monster Fish & River Monsters) – אנציקלופדיה מקיפה, אקדמית ואינטראקטיבית המוקדשת לתיעוד ולמחקר של דגי המים המתוקים הגדולים, המרתקים והעתיקים ביותר על פני כדור הארץ.

📌 אודות הפרויקט

הפרויקט נבנה מתוך אהבה לחקר הטבע (איכטיולוגיה) ונועד לספק מידע עמוק, מפורט וחסר פשרות על "מפלצות נהרות", שרידים פרהיסטוריים, ודיירי מיכלי זכוכית (Wet Pets).
האפליקציה פותחה בארכיטקטורה מודולרית מתקדמת, תוך שימוש ב-React ו-Tailwind CSS, ומותאמת ויזואלית באופן מושלם למסכי OLED מתקדמים (בפרט Samsung Galaxy S25 Ultra), עם עיצוב Deep Dark Mode מרשים וחסכוני בסוללה.

🧬 עומק מחקרי (Academic Standard)

כל דג באנציקלופדיה זו עבר תהליך מחקר קפדני הכולל:

פענוח שם ומורשת אטימולוגית: פירוק מלא של השם העממי, שם הסוג ושם המין (מיוונית/לטינית/שפות ילידיות).

חשיפה מדעית ונסיבות גילוי: רקע היסטורי על משלחות המחקר (במאות ה-18 וה-19) ומטרות החוקרים והזואולוגים (כגון לינאוס, קיווייה, ובולנג'ר).

שחר החיים (Juvenile Development): מסע ההישרדות הראשוני, אסטרטגיות רבייה והתפתחות הדגיגים.

דפוסים מעמקיים (Behavioral Ecology): אידאולוגיה הישרדותית, שיטות ציד מתוחכמות, והתנהגות טריטוריאלית.

פסגת האבולוציה: הישגים ביולוגיים, פיזיולוגיים ואקולוגיים, לצד חותם הזמן (תקופת הופעתם על פני כדור הארץ - מיוקן, קרטיקון, יורה ועוד).

🛠️ ארכיטקטורה ומבנה המאגר

המערכת בנויה בצורה מודולרית לחלוטין. קובץ ה-HTML הראשי שואב את הנתונים בזמן אמת מתוך קבצי נתונים נפרדים, מה שמונע עומס קוד ומאפשר עדכון קל של קטגוריות:

📂 giant-freshwater-fishes/
│
├── 📄 index.html                # מנוע הרינדור הראשי (React + Tailwind + UI)
│
└── 📂 data/                     # מסד הנתונים המחולק לקטגוריות
    ├── 📄 pool_legends.js       # האגם המפלצתי (אראפאימה, פאקו, אליגטור גאר ועוד)
    ├── 📄 glass_tanks.js        # מיכלי זכוכית (מבו, גוראמי ענק, דג ריאות)
    ├── 📄 world_monsters.js     # מפלצות עולם (ארוואנות, גוליית, צלופח חשמלי ועוד)
    └── 📄 ancient_survivors.js  # שורדים עתיקים (פדלפיש, חדקן, מסורן)


🚀 טכנולוגיות בשימוש

Frontend Framework: React (18.x) דרך CDN לרינדור דינמי ומהיר.

Styling: Tailwind CSS לעיצוב רספונסיבי, אנימציות עדינות ועיצוב Glassmorphism.

Icons: Lucide Icons.

Data Structure: JavaScript Objects (JSON-like) מחולקים לקבצים לשמירה על סדר וארגון.

📱 חווית משתמש (UI/UX)

חלוקה חכמה לקטגוריות: מניעת עומס קוגניטיבי על ידי הפרדת הדגים לבלוקים נושאיים במערך הגלריה.

חיפוש חי (Live Search): מנוע סינון חכם המחפש לא רק לפי שם הדג, אלא גם לפי שמו המדעי ומורשתו האטימולוגית.

תצוגת מודאל עוטפת: חווית קריאה נוחה ורציפה לכל ערך, המלווה בתגיות סטטוס שימור (IUCN), נתוני גודל, תוחלת חיים ותפוצה גיאוגרפית.

פרויקט זה נוצר ועודכן ברמת פירוט מקסימלית וללא פשרות, המוקדש לאהבת חקר מפלצות הנהרות ההיסטוריות של עולמנו.
